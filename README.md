# @inc2734/smooth-scroll

## Get started

### Install
```
$ npm install --save-dev @inc2734/smooth-scroll
```

### How to use

```
<script>
import { SmoothScroll } from '@inc2734/smooth-scroll';

new SmoothScroll(
  {
    selector: 'a[href^="#"]',
    offset: 100,
  }
);
</script>
```
