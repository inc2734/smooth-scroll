# @inc2734/smooth-scroll

## Get started

### Install
```
$ yarn add @inc2734/smooth-scroll
```

### Load scripts

```
<script src="/node_modules/@inc2734/smoothscroll/dist/smooth-scroll.js"></script>
```

### Setting

```
<script>
new SmoothScroll(
  {
    target: 'a[href^="#"]',
    offset: 100,
  }
);
</script>
```
